# Control de Nomina

Sistema de gestion de nomina, asistencia y personal para proyectos de construccion y obras.

## Descripcion

Este proyecto permite llevar el control administrativo de trabajadores en obra. Facilita el registro diario de asistencia, el calculo de horas extras, la gestion de anticipos de sueldo y la generacion de reportes de nomina.

## Caracteristicas Principales

- Gestion de Personal: Registro de trabajadores, cargos y areas.
- Control de Asistencia: Marcaje diario de asistencia y horas extras.
- Gestion de Anticipos: Registro y descuento de adelantos de sueldo.
- Calculo de Nomina: Generacion automatica de montos a pagar basado en dias trabajados.
- Reportes: Exportacion de datos a Excel (asistencias, nomina, listado de personal).

## Requisitos Tecnicos

- Java (JDK 8 o superior).
- Base de datos SQL Server.
- NetBeans IDE (recomendado para el desarrollo).

## Configuracion

La conexion a la base de datos se configura en la clase:
`src/main/java/conexsql/conexion.java`
